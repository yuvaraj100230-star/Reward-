Program Overview

This C++ program allows users to perform basic operations on cubic polynomials. A cubic polynomial has the form:

ax³ + bx² + cx + d

The program can:

  • Take coefficients for two cubic polynomials as input
  • Add the two polynomials
  • Subtract the second polynomial from the first
  • Display the results in polynomial form

Program Code

#include <iostream>
using namespace std;

int main() {
    int choice;

    // Coefficients for First Polynomial a1x^3 + b1x^2 + c1x + d1
    int a1, b1, c1, d1;

    // Coefficients for Second Polynomial a2x^3 + b2x^2 + c2x + d2
    int a2, b2, c2, d2;

    cout << "Enter the coefficients for FIRST Polynomial:\n";
    cout << "a1 = "; cin >> a1;
    cout << "b1 = "; cin >> b1;
    cout << "c1 = "; cin >> c1;
    cout << "d1 = "; cin >> d1;

    cout << "\nEnter the coefficients for SECOND Polynomial :\n";
    cout << "a2 = "; cin >> a2;
    cout << "b2 = "; cin >> b2;
    cout << "c2 = "; cin >> c2;
    cout << "d2 = "; cin >> d2;

    while (true) {
        cout << "\n=============================\n";
        cout << "  Polynomial Operations Menu\n";
        cout << "=============================\n";
        cout << "1. Addition of Polynomials\n";
        cout << "2. Subtraction of Polynomials\n";
        cout << "3. Exit\n";
        cout << "Enter your choice: ";
        cin >> choice;

        if (choice == 1) {
            // ADDITION
            int A = a1 + a2;
            int B = b1 + b2;
            int C = c1 + c2;
            int D = d1 + d2;

            cout << "\nAddition Result: "
                 << A << "x^3 + " << B << "x^2 + " << C << "x + " << D << endl;
        }
        else if (choice == 2) {
            // SUBTRACTION
            int A = a1 - a2;
            int B = b1 - b2;
            int C = c1 - c2;
            int D = d1 - d2;

            cout << "\nSubtraction Result: "
                 << A << "x^3 + " << B << "x^2 + " << C << "x + " << D << endl;
        }
        else if (choice == 3) {
            cout << "\nExiting program..." << endl;
            break;
        }
        else {
            cout << "\nInvalid choice! Please try again.\n";
        }
    }

    return 0;
}

Code Explanation

1. Variable Declaration

The program declares variables to store the coefficients of two cubic polynomials:

int a1, b1, c1, d1;  // Coefficients for the first polynomial
int a2, b2, c2, d2;  // Coefficients for the second polynomial

2. Input Collection

The program prompts the user to enter the coefficients for both polynomials:

cout << "Enter the coefficients for FIRST Polynomial:\n";
cout << "a1 = "; cin >> a1;
cout << "b1 = "; cin >> b1;
cout << "c1 = "; cin >> c1;
cout << "d1 = "; cin >> d1;

3. Menu System

A while(true) loop creates a menu system that continues until the user chooses to exit:

while (true) {
    // Display menu options
    // Process user choice
}

4. Polynomial Operations

For addition, the program adds corresponding coefficients:

int A = a1 + a2;  // Coefficient for x³ term
int B = b1 + b2;  // Coefficient for x² term
int C = c1 + c2;  // Coefficient for x term
int D = d1 + d2;  // Constant term

For subtraction, it subtracts the coefficients of the second polynomial from the first:

int A = a1 - a2;  // Coefficient for x³ term
int B = b1 - b2;  // Coefficient for x² term
int C = c1 - c2;  // Coefficient for x term
int D = d1 - d2;  // Constant term

5. Output

The results are displayed in polynomial form:

cout << "\nAddition Result: "
     << A << "x^3 + " << B << "x^2 + " << C << "x + " << D << endl;

Examples

Example 1: Addition

First Polynomial: 2x³ + 3x² + 4x + 5

Second Polynomial: 1x³ + 2x² + 3x + 4

Addition Result: 3x^3 + 5x^2 + 7x + 9

Example 2: Subtraction

First Polynomial: 5x³ + 4x² + 3x + 2

Second Polynomial: 1x³ + 2x² + 3x + 4

Subtraction Result: 4x^3 + 2x^2 + 0x + -2

Example 3: Negative Coefficients

First Polynomial: 3x³ - 2x² + 5x - 1

Second Polynomial: -1x³ + 4x² - 2x + 3

Addition Result: 2x^3 + 2x^2 + 3x + 2 Subtraction Result: 4x^3 + -6x^2 + 7x + -4
